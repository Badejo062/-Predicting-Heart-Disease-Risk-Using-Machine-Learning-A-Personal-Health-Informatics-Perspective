# Predicting-Heart-Disease-Risk-Using-Machine-Learning-A-Personal-Health-Informatics-Perspective
Machine learning analysis of clinical and behavioral health data to predict heart disease risk and demonstrate applications of predictive analytics within Personal Health Informatics and digital health monitoring systems.
This project aims to:
Identify key demographic and clinical predictors of heart disease
Evaluate machine learning models for cardiovascular risk prediction
Analyze feature importance to understand major health risk indicators
Demonstrate how predictive analytics can support digital health monitoring and Personal Health Informatics systems
